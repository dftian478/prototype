# 核心包

| 菜单 | 功能点 | 类型 | 逻辑描述 |备注|
|------|--------|------|------|------|
| 安装方式 | 部署 | radio | 默认选中部署 ||
| 包类型选择 | 核心包 | radio | 默认选中核心包 ||
| 架构选择 | 三种架构：x86、ARM、异构 | radio | 默认选中 x86 ||
| 打包记录 | 架构 | string | 三种架构中的一个：x86、ARM、异构 ||
| - | 已选功能 | string | 文案：核心功能。hover 后能有卡片展示核心功能点。 | |
| - | 生成时间 | string | 安装包的生成时间，也是默认的排序策略，按照生成时间倒序，最新的在上面。 ||
| - | 文件大小| string | 安装包的文件大小，单位：GB ||
| - | 清理状态 | string | 安装包的清理状态，状态为：永不清理。 ||
| - | 操作 | string | 操作按钮，文案：下载。单击后，展示弹框：<br>弹框中展示安装包的下载链接和MD5。签名文件的下载链接和MD5。其中下载链接支持三种下载方式：浏览器下载，第三方工具下载和命令行下载。||
