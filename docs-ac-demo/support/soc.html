Production Support Scope of Coverage

If we ship it, we...

Support

  • Installation
  • Usage
  • Configuration
  • Diagnosis
  • Bug reports (dependent on product life cycle)
  • Bug fixes

Do not support

  • Third-party software/drivers and uncertified hardware/hypervisors
  • Community projects upon which ACP releases are based
  • Code development
  • System and network design
  • Implementation and development of security rules and policies
